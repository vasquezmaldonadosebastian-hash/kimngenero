# Brief de Diseno - Recomendaciones Visuales Prioritarias

## Objetivo

Traducir el estado actual de KimnGenero en una lista de decisiones visuales prioritarias para el equipo de diseno grafico.

## Enfoque general recomendado

La propuesta visual deberia moverse en esta direccion:

- institucional pero no rigida
- contemporanea pero no de moda pasajera
- clara para lectura de datos
- sobria, confiable y facil de recordar

## Recomendaciones por tema

## 1. Jerarquia visual

### Lo que se observa hoy

- El sitio tiene buenos titulos y bloques claros.
- En algunas pantallas, mucha informacion compite en el mismo nivel.
- Las tarjetas del listado y la ficha de indicador pueden ordenar mejor sus capas visuales.

### Lo que se recomienda

- definir 3 niveles muy claros de titulo
- definir un estilo estable para etiquetas pequenas
- reducir ruido visual en bloques secundarios
- dar mas protagonismo a lo principal y mas silencio a lo accesorio

## 2. Sistema de color

### Lo que se observa hoy

- El azul institucional esta muy bien instalado.
- Existen varios colores tematicos para separar grandes grupos de indicadores.
- A veces esos colores pueden competir con la identidad principal.

### Lo que se recomienda

- mantener el azul como columna vertebral de marca
- definir una paleta secundaria mas ordenada para temas
- reservar colores intensos para destacar solo lo importante
- establecer reglas claras para:
  - fondos
  - botones
  - estados
  - etiquetas
  - categorias

## 3. Tarjetas del listado de indicadores

### Lo que se observa hoy

- Son la unidad visual mas importante del sitio.
- Contienen codigo, area, tema, titulo, descripcion, estado y boton.
- Funcionan, pero aun pueden verse mas refinadas y mas faciles de escanear.

### Lo que se recomienda

- simplificar el encabezado de cada tarjeta
- definir mejor la relacion entre codigo, categoria y titulo
- ordenar el peso visual del estado
- revisar el tamano del boton y su relacion con el resto del bloque
- estudiar una propuesta con mayor contraste entre informacion principal y secundaria

## 4. Ficha de indicador

### Lo que se observa hoy

- La pantalla tiene buen potencial visual.
- El encabezado es fuerte.
- La visualizacion principal necesita consolidarse como centro real de la experiencia.
- La informacion de metodologia y ficha tecnica debe sentirse mejor estructurada.

### Lo que se recomienda

- reforzar el encabezado como portada del indicador
- dar mas aire alrededor de la visualizacion
- ordenar el contenido tecnico en bloques mas legibles
- definir una relacion mas elegante entre la informacion analitica y la informacion de respaldo

## 5. Pantallas editoriales

Pantallas involucradas:

- Sobre el Modelo
- Glosario
- Contacto

### Lo que se observa hoy

- Tienen una base limpia y clara.
- Se sienten funcionales.
- Aun pueden ganar en ritmo editorial y personalidad visual.

### Lo que se recomienda

- mejorar la secuencia de lectura
- reforzar subtitulos y bloques intermedios
- trabajar citas, listados y cajas destacadas con mayor intencion visual
- hacer que estas pantallas se sientan parte del mismo universo que el resto del sitio

## 6. Visualizaciones incrustadas

Pantallas involucradas:

- Ficha de indicador
- Vista General
- Calendario

### Lo que se observa hoy

- Las visualizaciones son valiosas, pero no siempre se integran organicamente al resto de la pantalla.

### Lo que se recomienda

- mejorar el marco visual que las contiene
- reforzar titulos, notas y apoyos alrededor de la visualizacion
- hacer que la visualizacion parezca una parte natural del sistema y no un bloque externo insertado

## 7. Iconos y apoyos graficos

### Lo que se observa hoy

- Hay una mezcla entre iconos sobrios y recursos mas expresivos.

### Lo que se recomienda

- definir una sola familia visual de iconos
- establecer un criterio para simbolos decorativos
- decidir si el tono sera mas institucional o mas expresivo, pero evitar la mezcla sin control

## Entregables recomendados

## Minimo esperado

- paleta cromatica refinada
- jerarquia tipografica
- propuesta de tarjetas de indicadores
- propuesta de ficha de indicador

## Deseable

- lineamientos para pantallas editoriales
- criterio de iconografia
- reglas basicas de consistencia visual

## Ideal

- mini sistema visual con ejemplos de:
  - boton principal
  - tarjeta
  - bloque de titulo
  - etiqueta
  - estado
  - seccion editorial

## Orden sugerido de trabajo

1. Sistema de color y jerarquia tipografica
2. Tarjetas del listado
3. Ficha de indicador
4. Pantalla de inicio
5. Pantallas editoriales
6. Ajustes finales de consistencia

## Cierre

El sitio ya tiene una base buena y reconocible. El siguiente salto de calidad no depende de agregar mas elementos, sino de ordenar mejor la experiencia visual, reducir competencia entre senales y fortalecer la lectura de la informacion.
