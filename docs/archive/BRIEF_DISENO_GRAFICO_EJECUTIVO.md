# Brief Ejecutivo de Diseno - KimnGenero

## Que es KimnGenero

KimnGenero es un sitio institucional que muestra informacion sobre igualdad de genero por medio de cifras, fichas informativas y visualizaciones. Su objetivo es ordenar informacion relevante, volverla facil de consultar y transmitir confianza publica.

## Para que sirve este brief

Este documento resume lo esencial para que un equipo de diseno grafico pueda entender rapidamente:

- que tipo de sitio es
- cuales son sus pantallas mas importantes
- cual es su lenguaje visual actual
- donde estan hoy las mayores oportunidades de mejora

## Linea visual actual

### Colores principales

- Azul principal: `#0176DE`
- Azul oscuro: `#03122E`
- Azul oscuro secundario: `#173F8A`
- Azul claro: `#E8F2FF`
- Fondo claro: `#F5F4F8`
- Amarillo acento: `#FEC60D`

### Tipografia

- Montserrat para titulos y cifras destacadas
- Inter para textos de lectura y formularios

### Estilo general

- institucional
- moderno
- limpio
- orientado a informacion

## Lo que hoy funciona mejor

- La identidad institucional esta clara.
- Las pantallas principales se sienten ordenadas.
- Hay una buena base de color, aire y limpieza visual.
- La combinacion tipografica es solida.

## Lo que mas conviene redisenar

### 1. Tarjetas del listado de indicadores

Porque concentran mucha informacion y son la pieza mas repetida del sitio.

### 2. Ficha de indicador

Porque necesita mejorar la jerarquia entre encabezado, visualizacion y contenido de apoyo.

### 3. Sistema de color

Porque conviene ordenar mejor la relacion entre color institucional y colores tematicos.

### 4. Pantallas editoriales

Porque necesitan una lectura mas refinada en titulos, bloques y ritmo visual.

## Entregables esperados del equipo de diseno

- propuesta de linea grafica general
- paleta cromatica refinada
- jerarquia tipografica
- rediseno del listado de indicadores
- rediseno de la ficha de indicador
- reglas visuales para pantallas editoriales
- criterio unificado de iconos y apoyos graficos

## Prioridad sugerida

1. Inicio
2. Listado de indicadores
3. Ficha de indicador
4. Sistema visual general
5. Pantallas editoriales

## Cierre

KimnGenero no necesita inventar una identidad desde cero. Ya tiene una base institucional clara. El mayor aporte del diseno esta en darle mas orden, mas presencia y una mejor jerarquia visual a la informacion.
