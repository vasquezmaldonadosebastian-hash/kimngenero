# Brief de Diseno - KimnGenero (Version con Screenshots)

## Resumen del Proyecto

KimnGenero es una plataforma institucional dedicada a mostrar informacion sobre igualdad de genero por medio de cifras, visualizaciones y fichas informativas. Su proposito es hacer visible informacion relevante de manera clara, confiable y ordenada.

## 5. Linea Visual Actual

### Colores principales

- Azul principal: `#0176DE`
- Azul oscuro: `#03122E`
- Azul oscuro secundario: `#173F8A`
- Azul claro: `#E8F2FF`
- Fondo gris claro: `#F5F4F8`
- Amarillo acento: `#FEC60D`

### Tipografia

- Montserrat para titulos y cifras destacadas
- Inter para textos de lectura

### Estilo general

- fondos suaves
- bloques blancos
- esquinas redondeadas
- sombras ligeras
- encabezados fuertes
