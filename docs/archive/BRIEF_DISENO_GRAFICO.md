# Brief de Diseño - KimnGenero

## Resumen del Proyecto

KimnGenero es una plataforma institucional dedicada a mostrar información sobre igualdad de género por medio de cifras, visualizaciones y fichas informativas. Su propósito es hacer visible información relevante de manera clara, confiable y ordenada, para apoyar la comprensión pública y la toma de decisiones.

Visualmente, el sitio ya tiene una identidad bastante definida: mezcla un tono institucional serio con una presentación moderna, limpia y orientada a datos. Aun así, hay espacio para fortalecer la jerarquía visual, unificar criterios entre pantallas y hacer que la experiencia se sienta más sólida y memorable.

## Objetivo de este Brief

Entregar al equipo de diseño gráfico una lectura clara del sitio actual para que pueda proponer mejoras visuales sin tener que interpretar código ni documentación técnica.

La intención no es rediseñar desde cero, sino entender:

- qué pantallas existen
- qué rol cumple cada una
- qué lenguaje visual usa hoy el sitio
- qué piezas sería valioso redefinir o fortalecer

## Mapa del Sitio

Contiene:

- un bloque final de invitación a explorar el contenido

Rol visual:

## Estructura General de la Interfaz

1. franja institucional superior

4. contenido central
5. pie de página institucional amplio

Esto genera una sensación de orden y continuidad. Para diseño, esto significa que hay una buena base para consolidar un sistema visual consistente entre pantallas.

## Paleta principal

Los colores más relevantes encontrados en el sitio son:

- **Azul principal:** `#0176DE`
- **Azul oscuro profundo:** `#03122E`
- **Azul oscuro secundario:** `#173F8A`
- **Azul muy claro:** `#E8F2FF`
- **Fondo gris claro:** `#F5F4F8`
- **Amarillo acento:** `#FEC60D`
- **Blanco:** como base de tarjetas, formularios y bloques de lectura

Lectura visual de esta paleta:

- el azul principal funciona como color institucional y de acción
- el azul oscuro da seriedad y peso
- el azul muy claro suaviza fondos y zonas de apoyo
- el amarillo aparece como acento puntual
- el blanco da limpieza y respiración

Además, en el listado de indicadores se usan colores complementarios para distinguir grandes temas. Hoy aparecen tonos como:

- rojo
- verde
- violeta
- celeste
- ámbar
- rosado

Esto ayuda a separar categorías, pero también abre una oportunidad: ordenar mejor la lógica cromática para que no compita con el azul institucional.

## Tipografía

La identidad tipográfica actual se organiza en dos niveles:

- **Montserrat** para títulos, cifras destacadas y encabezados
- **Inter** para textos de lectura, formularios y descripciones

Lectura visual:

- Montserrat aporta fuerza, presencia y un tono institucional contemporáneo
- Inter aporta claridad y buena lectura en textos más largos

Es una combinación sólida y funcional. El trabajo del diseñador podría enfocarse más en la jerarquía, tamaños, pesos y ritmos que en cambiar completamente la dupla tipográfica.

## Estilo de íconos

El sitio usa principalmente:

- íconos lineales, simples y limpios
- algunos símbolos gráficos muy directos
- en ciertas zonas, pequeños íconos ilustrativos o expresivos

Lectura visual:

- la línea general tiende a lo sobrio y funcional
- hay una pequeña mezcla entre lenguajes más institucionales y otros más expresivos

Esto sugiere una oportunidad clara: definir una sola familia visual para íconos y apoyos gráficos, de manera que el sitio se vea más unificado.

## Observaciones Visuales Relevantes para Diseño

## Lo que ya funciona bien

- La identidad azul está clara y es reconocible.
- Hay una base visual limpia y ordenada.
- Los bloques blancos sobre fondos suaves ayudan a separar información.
- La combinación tipográfica tiene buen potencial institucional.
- Las pantallas principales ya muestran una intención visual coherente.

## Lo que conviene revisar

- La relación entre color institucional y colores temáticos.
- La jerarquía de las tarjetas del listado.
- El protagonismo real de las visualizaciones dentro de la ficha de indicador.
- La consistencia entre pantallas editoriales y pantallas de consulta.
- La unificación del estilo de íconos y apoyos gráficos.
- El tratamiento de títulos, subtítulos y textos largos en pantallas densas como “Sobre el Modelo” y “Glosario”.
- La integración visual del calendario y de otras visualizaciones incrustadas.

## Entregables Esperados del Diseñador

Se espera que el diseñador entregue una propuesta visual orientada a fortalecer la claridad, la consistencia y la presencia institucional del sitio.

### 1. Propuesta de línea gráfica general

- definición de tono visual
- criterios de estilo para el sitio completo
- referencias de atmósfera, seriedad y modernidad

### 2. Paleta cromática refinada

- paleta principal
- paleta secundaria
- colores de apoyo
- propuesta para diferenciar grandes temas sin perder unidad visual

### 3. Jerarquía tipográfica

- propuesta de tamaños
- pesos
- reglas de uso para títulos, subtítulos, cifras, etiquetas y textos largos

### 4. Rediseño del listado de indicadores

- nueva propuesta para las tarjetas
- mejor orden de información
- tratamiento de estados, categorías y botones

### 5. Rediseño de la ficha de indicador

- mejor distribución del encabezado
- nueva jerarquía para la visualización principal
- tratamiento visual de metodología y ficha técnica

### 6. Criterios visuales para pantallas editoriales

- propuesta para “Sobre el Modelo”
- propuesta para “Glosario”
- reglas para bloques de texto, citas, listados y llamados de atención

### 7. Sistema de íconos y apoyos gráficos

- criterio unificado para íconos
- estilo para símbolos, etiquetas, acentos y pequeños recursos ilustrativos

### 8. Guía breve de uso visual

- una guía resumida para mantener consistencia
- ejemplos de aplicación en botones, tarjetas, fondos, títulos y bloques informativos

## Prioridades Recomendadas

Si el trabajo debe hacerse por etapas, se recomienda este orden:

1. Inicio
2. Listado de indicadores
3. Ficha de indicador
4. Línea gráfica general
5. Pantallas editoriales
6. Pantallas secundarias

## Cierre

KimnGenero ya tiene una base visual seria, limpia y con vocación institucional. El desafío no es inventar una identidad desde cero, sino convertir esa base en un sistema gráfico más claro, coherente y memorable.

El mayor valor que puede aportar diseño gráfico aquí está en:

- ordenar la lectura
- dar más fuerza a la información prioritaria
- hacer más consistente la experiencia entre pantallas
- elevar la calidad percibida del observatorio
