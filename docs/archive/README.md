# Archivo documental

Esta carpeta contiene material auxiliar, historico o de trabajo que no debe usarse como fuente principal para decisiones tecnicas del proyecto.

## Contenido esperado

- briefs de diseno
- borradores o reportes historicos
- documentos de apoyo no normativos

## Regla de uso

Si un documento esta en `docs/archive/`, debe contrastarse siempre con:

- `docs/ARCHITECTURE.md`
- `docs/API_SURFACE.md`
- `docs/TEST_STRATEGY.md`
- `docs/PRODUCTION_CONFIGURATION.md`
- `docs/PLAN_SANEAMIENTO_REAL.md`
